# Smart Attendance System ERP - MVP Scaffold

This repository contains the initial scaffold for the Smart RFID Attendance Management System (PACR NEXUS)

Includes:
- Next.js + TypeScript + Tailwind CSS
- Design tokens (CSS variables)
- Core components (Sidebar, TopNav, KPI, ChartCard, Table, Modal, Toast)
- Mock data & sample charts
- Next.js pages router scaffold
- Mock notification sender email: asvanthasvanth008@gmail.com

Quick start
1. git clone git@github.com:ASVANTH1519/smart-attendance-system-erp.git
2. git checkout erp/mvp-scaffold
3. npm install
4. npm run dev

Replace placeholder assets in /public as needed and follow README details for Storybook and Vercel.
